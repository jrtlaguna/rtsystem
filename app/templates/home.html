Welcome!

This is the Employee Accomplishment Report System of Quezon City - Environmental Protection and Waste Management Department.

How to use the system:



HOW TO CREATE A NEW ACCOMPLISHMENT REPORT
 For Attendance:
  1. Select an attendance period.
  2. In case of absences and/or holidays:
    1. Select a date on the calendar.
    2. Choose the corresponding reason for absence for that day (e.g. absent, holiday, etc.).
    3. Repeat steps for each absence/holiday.
  3. Once done, click 'Next.'
 For Tasks:
  1. Select a specific week.
  2. Select all tasks accomplished for the week.
    • For regular tasks, tick the checkboxes before the task.
    • For tasks with quantity, input the quantity in the box before the task.
  3. If task is not yet on the list:
    1. Click 'Add Task.'
    2. Enter task description.
    3. Select the week it applies to.
      • If the task is applicable for multiple weeks/months, turn on Recurring Task.
    4. If the task involves quantity, turn on Has Quantity.
    5. Click 'Add'.
  4. Once all tasks accomplished for the week have been selected, click 'Next'.
 For Review:
  1. Review the report.
  2. Once the report is confirmed, click 'Submit'.
    • The accomplishment report will be sent to your designated Section/Division Chief for approval.
HOW TO VIEW AND PRINT ACCOMPLISHMENT REPORTS
 To view and edit accomplishment reports:
  1. Select the accomplishment report you want to view.
    • Each report has a corresponding status: Pending or Approved.
  2. To edit a report:
    1. Click 'Edit'.
    2. Edit all necessary details.
    3. Note: By clicking 'Submit', the report will be sent to your Section/Division Chief for re-approval.
  3. To view changes made to your reports, click 'Change Log'.
 To print an approved accomplishment report:
  1. Select the accomplishment report you want to print.
  2. Click 'Print'.
HOW TO VIEW YOUR PROFILE AND LOG OUT
 To view profile:
  • Select 'My Profile'.
    • If changes are necessary, contact the Administrative Division.
 To log out from your account:
  • Select 'Logout'.

How to use the system:



HOW TO APPROVE ACCOMPLISHMENT REPORTS
  1. Select an employee to view their report.
  2. Review the validity of tasks accomplished by the employee.
  3. If errors within the report are found:
    • Click 'Edit'.
    • Edit necessary information.
    • Click 'Save'.
  4. Click 'Approve' to change status of report from "Pending" to "Approved"
    • Report will be sent back to the employee for printing and signing.

HOW TO MANAGE ALL USER ACCOUNTS
 To create a new account:
  1. Click '+ New'.
  2. Select an account type.
  3. Enter account information.
  4. Click 'Create'.
 To edit a profile:
  1. Select an account type.
  2. Select name of user to view their profile.
  3. Click 'Profile'.
  4. Edit necessary information.
    • To deactivate an account: Change status from Active to Inactive
      • Note: Inactive accounts are transferred to the 'View Archive' page.
  5. Click 'Save'.
 To view attendance:
  1. Select account type 'Employee'.
  2. Select a section/division.
  3. Select an employee and click 'Attendance' to be redirected to their attendance page.
 To view accomplishment reports:
  1. Select account type: Employee.
  2. Select a section/division and employee.
  3. Click 'Reports' to be redirected to their reports page.
  4. Select the accomplishment report you want to view.
    • Each report has a corresponding status: Pending or Approved.
    • To delete a report, click 'Delete'.
    • To print an approved accomplishment report, click 'Print.'
  5. To view changes made to the reports, click 'Change Log'.
HOW TO MANAGE POSITIONS
 To create a new position:
  1. Click '+ New'.
  2. Select: Position.
  3. Select section/division of new position.
  4. Input position title.
  5. If employee has generic primary tasks per week/month:
    1. Click .
    2. Input task description.
  6. Click 'Create'.
 To create a new section/division:
  1. Click '+ New'.
  2. Select: Section/Division.
  3. Input Section/Division Name.
  4. Select user assigned to verify reports for the new section/division.
  5. Click 'Create'.
 To edit position details:
  1. Click the position title.
  2. Edit all necessary details.
    • To delete a position, click 'Delete'.
  3. Click 'Save'.
 To edit section/division details:
  1. Click the section/division name.
  2. Edit all necessary details.
    • To delete a section/division, click 'Delete'.
  3. Click 'Save'.
HOW TO MANAGE INACTIVE ACCOUNTS
 To reactivate account:
  1. Select an account.
  2. Click 'Profile'.
  3. Change status from Inactive to Active
    • Report will return to 'Manage Accounts' page.
 To delete an account:
  1. Select an account.
  2. Click 'Delete'.
HOW TO VIEW YOUR PROFILE AND LOG OUT
 To view profile:
  • Select 'My Profile'.
    • If changes are necessary, go to 'Manage Accounts' page and edit the profile details.
 To log out from your account:
  • Select 'Logout'.
